## 商店

ps:代名词√[此符号意思为:你鼠标所指的商店]

手上拿着对应物品点击箱子创建商店[并且在聊天栏输入你要出售的价格]

| /qs buy    | √商店换成收购 |
| ---------- | ------------- |
| /qs sell   | √出售         |
| /qs remove | √删除商店     |

右键√商店可进行商店编辑[聊天框中点击即可]
